# pdf2booklet
Convert a single-page PDF to booklet format using pdflatex and pdfpages.

# Requirements

- pdflatex
- pdfpages LaTeX package

# Usage

- Place your PDF in `bin/input.pdf`
- Run `make booklet` in Terminal
- pdflatex generates `bin/booklet.pdf`
